Validador de Importadores SQL en Tiempo Real

Monitoreo de Datos VBA · Excel · SQL Fuentes de Red

Desarrollé el Validador de Importadores, una herramienta en VBA que monitorea en tiempo real el estado de actualización de múltiples fuentes de datos en red. El sistema compara la fecha de modificación de cada carpeta origen con la fecha actual, clasifica el estado como 'Actualizado', 'Falta Actualizar' o 'ERROR', y genera un dashboard visual en Excel con indicadores de color. Esto permite al equipo de datos detectar fallas de importación sin necesidad de revisar manualmente cada fuente.

Dashboard de monitoreo. El panel central muestra el estado de cada fuente de datos con indicadores de color —verde para actualizado, amarillo para pendiente, rojo para error—. De un vistazo, el analista puede identificar qué importadores requieren atención sin abrir ninguna carpeta.

Macro de comparación de fechas. La función usa FileSystemObject para obtener la fecha de última modificación de la carpeta en la ruta de red. Si la fecha coincide con la fecha actual en día, mes y año, escribe 'Actualizado' y activa un flag VERDADERO; en caso contrario, escribe 'Falta Actualizar'.

Lógica de detección de errores. Una segunda macro verifica si una carpeta fue modificada hoy. Si la modificación fue hoy y la carpeta debería estar vacía o no modificarse, la macro señala el estado como 'ERROR', alertando una posible corrupción o importación no solicitada.

Macro de apertura de archivo de referencia. Para la validación cruzada, la macro abre automáticamente el archivo Excel de referencia en la ruta de red configurada, accede a sus datos y los compara con los registros locales, asegurando integridad entre la fuente y el destino.

Registro de auditoría. Cada ejecución del validador escribe una entrada en el log de auditoría con marca de tiempo, nombre de la fuente y estado detectado. Esto permite rastrear el historial de actualizaciones, identificar patrones de falla y generar reportes de disponibilidad para el equipo de TI.
