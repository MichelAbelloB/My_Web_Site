Automatización de procesos

Contrato Propiedad Horizontal

Para la realización de contratos de propiedades horizontal se creó un proceso automático que auto completa las cajas de texto del formato establecido. Todo ello mediante un formulario de Word el cual interactúa con todas las cajas de texto y mediante las macros se verifica la celda y se auto completa la información según la información colocada en las celdas, para ser dispuesta en la ubicación relativa según su la disposición de la hoja de Word.

El objetivo de este formato era estandarizar en un inicio el formato que se debía tomar según las implicaciones legales de la normativa colombiana y como segundo un formato estructurado que permitiera ser legible y escalable en el tiempo. Por ello, primero se tuvieron en cuenta estas disposiciones y después se realizó el proceso de automatización según los bloques funciones que debían ser auto completados con las macros y visualizados por el formulario.

Dentro del apartado final del contrato se estableció un botón de “Actualización Contrato”, el cual tiene como principal función mostrar el formulario creado con los respectivos espacios. Estos espacios como ya se ha mencionado nos permitirá mediante los bloques de texto recibir información y una vez dentro del formulario “ACTUALIZAR CONTRATO” cargar los datos dentro del documento con la macro. Todo ello con una interfaz sencilla que permite diferenciar el arrendatario del arrendador y su correspondiente información personal.

Para el establecimiento de la macro se definió un ancho predefinido de 1 según el insertado de datos en la hoja de Word. Para el ingreso de datos de cada una de las celdas de texto se creó variables definidas según el rango. Teniendo esto como base se establecerá un set para cada una de las variables y se estructurará la activación del documento de cada variables dentro del documento.

Una vez se encuentre activo el documento se realizará una concatenación de las variables con los marcadores de la ubicación relativa según el nombre definido para el marcador. Una vez establecidos los parámetros de las variables igualaremos las variables con los bloques de texto. De esta maneja los datos dispuestos en las celdas de texto serán dispuestos en las posiciones relativas en el documento y se realizara una limpieza de la consola creada con el formulario.
