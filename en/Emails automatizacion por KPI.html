Automation of sending KPIs by email

The Excel interface was designed to facilitate the entry and organization of data necessary to generate emails with KPIs and send them to specific recipients. The email addresses of the primary and copy recipients are entered in cells E3 and E4, respectively. The KPI values are distributed into specific cells, such as E8, E7, E6, E9, F8, F7, F6, F9, G8, G7, G6, and G9, according to the VBA code. These cells store data such as Payment Promised Percentage (PP), Payment Taken Percentage (PT), Conversion Rate (CR), among others, which is then extracted to construct the email content. In this way, the interface in Excel provides a structured and efficient way to enter the information required for automating the email sending process with KPIs.

To automatically send an email through Outlook, the body of the email is constructed in HTML format, including images, text and HTML tables that contain information about the KPIs, which will be linked to an SQL search and other relevant details. Once the email body has been constructed, it is assigned to the email object and sent to the recipients specified within cells B2 and B3 in excel.

In this code snippet, you are building a macro in Visual Basic for Applications (VBA) that automates the process of generating and sending emails through Microsoft Outlook using data from an Excel spreadsheet. The macro begins by declaring variables to store Outlook-related objects, such as the email recipients, the message content, and the Outlook application itself. These variables will be used later in the macro.

The Outlook application is then instantiated using the CreateObject("Outlook.Application") function. This allows the code to interact with Outlook and perform actions such as creating and sending emails. Next, a new email object is created using objOutlook.CreateItem(olMailItem). This object will be used to build the email that will be sent later. The olMailItem constant specifies that a new email is being created, rather than another type of Outlook item.

The email addresses of the recipients and those in copy are obtained from specific cells of the Excel spreadsheet and stored in corresponding variables. This ensures that the email will be sent to the correct addresses. Using the DateAdd function, the date of the previous day is calculated. This involves subtracting a day from the current date and the result is stored in a variable for later use in the email. The email subject is constructed by concatenating fixed text with the formatted current date and other specific details. The subject provides a brief description of the content of the email, allowing recipients to quickly understand its purpose and relevance.

Values are then assigned to variables related to Key Performance Indicators (KPIs) using data from specific cells in the Excel spreadsheet. These values are formatted as necessary and will be used to include relevant information in the body of the email. Finally, the body of the email is constructed in HTML format, including images, text, and HTML tables containing information about the KPIs and other relevant details. Once the email body has been constructed, it is assigned to the email object and sent to the specified recipients.
