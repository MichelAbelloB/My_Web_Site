Process automation

Horizontal Property Contract

To carry out horizontal property contracts, an automatic process was created that auto-completes the boxes. text of the established format. All this through a Word form which interacts with all the text boxes and through the macros the cell is verified and the information is auto-completed according to the information placed in the cells, to be arranged in relative location according to your Word sheet layout.

The objective of this format was to initially standardize the format that should be taken according to the implications legal provisions of Colombian regulations and secondly, a structured format that would allow it to be readable and scalable. in the time. Therefore, these provisions were first taken into account and then the process of automation according to the function blocks that had to be auto-completed with the macros and visualized by the form.

Within the final section of the contract, an “Update Contract” button was established, which has the purpose of main function to show the form created with the respective spaces. These spaces, as already mentioned It will allow us to receive information through the text blocks and once inside the form “UPDATE CONTRACT” load the data into the document with the macro. All this with a simple interface that allows you to differentiate the lessor's tenant and their corresponding personal information.

To establish the macro, a predefined width of 1 was defined according to the data inserted in the Word sheet. To enter data from each of the text cells, variables defined according to the range were created. having this As a basis, a set will be established for each of the variables and the activation of the document for each will be structured. variables within the document.

Once the document is active, a concatenation of the variables with the location markers will be carried out. relative according to the name defined for the marker. Once the parameters of the variables have been established, we will equalize the variables with text blocks. From this handle the data arranged in the text cells will be arranged in the relative positions in the document and a cleaning of the console created with the form will be carried out.
