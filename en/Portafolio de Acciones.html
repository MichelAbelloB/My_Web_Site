Analysis of data

Stock Portfolio

For the creation of the investment portfolio. It was divided into variable income investments and equity investments. fixed rent; In this way, an attempt was made to categorize the type of products available in each category by their relevance ranking from highest to lowest respectively. In addition to incorporating functions that allow the entry of new data, the export of existing data within the table and the analysis of data according to the defined categorization.

Initially, the data entry interface for the fixed income investment portfolio is created, for this The main interaction functionalities that allowed deleting and entering records were determined. taking into account nature to collect information. As a complementary tool, a button is created guide, where relevant information is available on the types of classification taking into account the existing ones in the market.

For the creation of the input interface and interaction tools. Macros were created for collection information of the available cells, the clean macro is created to eliminate the temporary records in the data cells, the save information macro is created, which allows saving the available records within the data cells. data, as well as clearing the data cells and generating a save success notification if the save conditions filling and data type are met.

Once the data is received through the fixed income portfolio entry interface. Data is stored within the table “CAPITAL MARKET INVESTMENT PORTFOLIO” in this table is generated automatically with the entry interface the interaction of entry and deletion of data according to search interval.

To create the entry interface for the variable income investment portfolio, the import is carried out of data from thebvc (Colombian Stock Exchange). In this case, the 20 most relevant investments from January 1 are taken from 2021 to March 31, 2021. Keeping this information in mind, to create the home interface, designs an interaction interface very focused on links by type of investment. Plus adding a guide button that presents the most relevant information for the generation of investment reports.

Within each of the tabs designed for the analysis of investment data, a model was established where The information and its respective analysis were divided according to the indicators, results and graphs. Each of These tabs are linked to an interaction menu at the top of each analysis and a small summary of each company.

Within the indicators we find the most relevant indicators according to the standard at that time analyzed. Ecopetrol. Furthermore, to be a homogeneous measure, all companies were analyzed with the same indicators and the same capital invested.

For the graphics section, dynamic tables were established linked to the results section where two were designed. graphics; a linear one where the closing price was determined during this time interval and an analysis was performed linear trend to determine the closing price trend. Furthermore, in the second graph a analysis using the candlestick chart where the decreases and increases within the model were determined according to the Time interval.
