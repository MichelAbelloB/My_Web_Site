Teleperformance Analysis

To carry out the Teleperformance analysis, a general analysis of the primary data was carried out by agent. For this, the satisfaction index was determined according to surveys received per user according to the receiving agent. In addition to cross-referencing information according to the supervisor by agent, length of seniority and lob; All this allowed us to determine if there were promoters, detractors, neutrals, solutions or non-solutions within the approach to the final report.

To establish the key indicators (KPI), the NPS was used, which calculates the participation of a question according to its rating between the range of 0 to 10 according to user satisfaction. As second As a measure, the AHT or the average time taken per agent was used (this measure is very common within the call center metrics). As a third measure, the percentage of non-participation, which refers to the number of surveys that were not qualified according to the overall number of qualified surveys. Finally, we have the participation percentage, which calculates the number of calls per agent over the total number of calls.

To prepare the first analysis, a scatter diagram was created where the AHT indicator evolved. vs the percentage of participation. For this, a linear trend was determined, which when analyzed is evidence that there is no correlation between these two variables evaluated.

Taking into account the percentage of participation of each agent and the average time per service between intake per agent vs the number of surveys received. We can determine that there is no correlation between these variables, since, although the participation of the agents is linked to the calls received, there is no relationship plausible showing that the number of calls affects participation, in correlation with AHT.
